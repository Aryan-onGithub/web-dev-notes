some imp core web vitals

  1. CLS (cumilative layout shift)
    ideal: the layout of the website should not shift when the page is loading
  2. LCP (larges contentful point)
    ideal: largest and imp piece of content in the page shoud take less the 2.5 sec to load
  3. FIC (first input delay)
    ideal: you input should be processed under 100ms
