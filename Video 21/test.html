cascade algo tells us:

  1. last selector will win if all are of same type
  2. specifity: inline style > id selector > class or attribute selector > element selector > universal selector
  3. origen: if no style is applied then some default style from browser will be used, but can be overide with any selectors
  4. !important: if you write !important after any selectors, then their porperty will be applied
    eg:- color: red!important;
